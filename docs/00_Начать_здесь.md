# Начать здесь

Эта папка содержит простую документацию по установке, запуску и деплою проекта.

Проект состоит из двух основных частей:

- `backend/` - локальный API-сервер на Node.js.
- `admin-panel/` - frontend/admin-panel на React + Vite.

Дополнительно backend может использовать Python-зависимости для Fox/Camoufox и Playwright.

## Быстрый старт

### 1. Установить базовые инструменты

Нужно установить:

- Node.js: рекомендуется Node.js 20 LTS или новее.
- npm: устанавливается вместе с Node.js.
- Python 3: рекомендуется Python 3.10 или новее.
- pip: устанавливается вместе с Python или отдельно через официальный установщик.

### 2. Установить backend

```bash
cd backend
npm ci
npm run playwright:install
python3 -m pip install -r requirements-fox.txt
```

Если `npm ci` не подходит, используйте:

```bash
npm install
```

### 3. Установить admin-panel

```bash
cd ..
cd admin-panel
npm ci
```

Если `npm ci` не подходит, используйте:

```bash
npm install
```

### 4. Запустить backend

```bash
cd backend
npm run dev
```

По умолчанию backend слушает:

```text
http://localhost:3000
```

Проверка:

```text
http://localhost:3000/api/health
```

### 5. Запустить admin-panel

В другом терминале:

```bash
cd admin-panel
npm run dev
```

Vite покажет локальный адрес, обычно:

```text
http://localhost:5173
```
