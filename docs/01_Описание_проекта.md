# Описание проекта

Этот документ описывает только технический запуск и деплой. Внутренняя бизнес-логика проекта здесь не раскрывается.

## Структура репозитория

```text
/
  backend/
  admin-panel/
  vercel.json
  docs/
```

## `backend/`

`backend/` - локальный API-сервер на Node.js.

Основные файлы:

- `backend/package.json` - npm-скрипты и Node.js-зависимости backend.
- `backend/package-lock.json` - зафиксированные версии npm-зависимостей.
- `backend/requirements-fox.txt` - Python-зависимости для Fox/Camoufox-моста.
- `backend/src/server.js` - точка запуска API-сервера.

Скрипты из `backend/package.json`:

```bash
npm run dev
npm start
npm test
npm run playwright:install
```

Основные Node.js-зависимости backend из `backend/package.json`:

- `express` `^4.21.2`
- `better-sqlite3` `^11.7.0`
- `playwright` `^1.59.1`

Python-зависимости backend из `requirements-fox.txt`:

- `camoufox>=0.4.11`
- `playwright>=1.40.0`

## `admin-panel/`

`admin-panel/` - frontend/admin-panel на React + Vite.

Основные файлы:

- `admin-panel/package.json` - npm-скрипты и frontend-зависимости.
- `admin-panel/package-lock.json` - зафиксированные версии npm-зависимостей.
- `admin-panel/vite.config.ts` - настройки Vite, dev/preview proxy и base path.
- `admin-panel/src/api/client.ts` - настройка URL для API.

Скрипты из `admin-panel/package.json`:

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

Основные зависимости admin-panel из `admin-panel/package.json`:

- `react` `^19.2.4`
- `react-dom` `^19.2.4`
- `react-router-dom` `^7.14.0`
- `vite` `^8.0.4`
- `typescript` `~6.0.2`
- `tailwindcss` `^4.2.2`

## `vercel.json`

В корне репозитория есть `vercel.json`.

Он указывает Vercel:

```json
{
  "installCommand": "cd admin-panel && npm ci",
  "buildCommand": "cd admin-panel && npm run build",
  "outputDirectory": "admin-panel/dist"
}
```

Это значит, что Vercel должен собирать frontend из папки `admin-panel/`.
